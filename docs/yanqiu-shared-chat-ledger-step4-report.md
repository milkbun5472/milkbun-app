## 本步实际开通

### CC 写入

- MCP 新增 `append_cc_turn`。
- 只允许固定言秋/小克身份，拒绝借该工具写其他角色。
- 调用方（言秋本人）按句段选择 `life / emotion / decision / joke`；纯施工句段不回流。
- 句段只准逐字删节，不准总结、改写、推测或写隐藏思考。
- `session_id + turn_id + side + segment_index` 生成稳定幂等键；重试不复制，同文不同轮不合并。
- 同一幂等键若出现不同正文，拒绝覆盖并报错。
- 多句段使用一次批量 INSERT；不写、不回退整份 `saves`。

### App 影子拉取

- 只拉 `source in (cc, stackchan)` 且只限言秋 ID。
- 使用 `(updated_at, id)` 复合游标，后补软删也能观察。
- 启动、回前台、恢复联网及每分钟触发。
- 本步只保存无正文诊断：累计行数、软删数、重复键、同文不同轮、发生时间乱序和错误。
- 不保存入站正文、不合并言秋私聊、不进入模型 prompt。
- 拉取失败不推进游标；换账号重建独立观察状态。
